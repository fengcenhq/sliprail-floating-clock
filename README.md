# sliprail-floating-clock

A floating clock window that always stays on top of other applications. Displays the current time with precise seconds, perfect for time-sensitive tasks and monitoring.

## Install Extension

Click the button below to install this extension directly in Sliprail:

[![Install in Sliprail](https://img.shields.io/badge/Install_in_Sliprail-Click_Here-blue?style=for-the-badge)](https://sliprail.fengcen.io/install-extension?id=5f2ff6bf-ed0b-4cbf-a00a-9c466c663d27)

**Note:** If Sliprail is not installed on your computer, the button will not work. Please download and install Sliprail from the [official website](https://sliprail.fengcen.io) first.

## Features

- **Always on Top**: Keeps the time visible over full-screen apps and other windows.
- **Precise Timing**: Shows hours, minutes, and seconds.
- **Draggable UI**: Move the clock anywhere on your screen.
- **Minimalist Design**: Transparent background with rounded corners.
- **Single Instance**: Ensures only one clock window is open at a time.

## Getting Started

1. Install the extension using the button above or via the Sliprail Store.
2. Use the shortcut "Open Floating Clock" in Sliprail to launch the widget.
3. Drag the clock to your desired position.
